# QueryGen
QueryGen is an AI-powered Natural Language to SQL Generator that converts plain English questions into SQL queries and executes them on a PostgreSQL database.

Features

* Natural Language to SQL conversion using Gemini
* FastAPI backend for query processing
* PostgreSQL database integration
* Interactive frontend built with HTML, CSS, and JavaScript
* Query history support
* Dynamic SQL execution and result visualization
* Protection against unsafe SQL operations

Tech Stack

* Python
* FastAPI
* PostgreSQL
* Gemini API
* HTML, CSS, JavaScript

Example

Input

Show top 5 customers by spending

Output

* Generates SQL automatically
* Executes query on PostgreSQL
* Returns results in a table format
